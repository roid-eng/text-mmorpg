# CLAUDE.md — Text MMORPG

## Text MMORPG 기획

### 시장 조사 요약
- 글로벌 텍스트 MMORPG 트렌드 지수: 42.5 (틈새, 꾸준한 수요)
- 한국 트렌드 지수: 0 (미개척)
- 벤치마크: CyberCode Online (평점 4.8, 100만+ 설치)
- 타겟: 글로벌 (한국 포함, 영문/한국어 선택 가능)
- 핵심 성패 요인: UI/UX, 온보딩, 콘텐츠 깊이

### 확정된 방향
- 장르: 텍스트 MMORPG
- 핵심: 글로 만드는 몰입감 (텍스트 RPG 감성)
- 스타일: 고전 MUD 감성
- 세계관: 정통 판타지
- 콘텐츠 루프: 사냥 + 탐험 + 기타 혼합
- 플레이어: 평범한 인간으로 시작 → 성장
- 멀티방식: 비실시간

### 세계관 — Mytharion
- 분위기: 신비로운
- 지형: 단일 대륙
- 핵심 설정: 고대 신들의 전쟁 → 봉인/소멸
             흔적이 세계 곳곳에 산재
- 지역 구성: 자연 환경 기반 + 그 안에 세력 존재

### 캐릭터 시스템
- 직업: 전사 / 마법사 / 궁수 / 성직자 (고정 직업제)
- 스탯: STR / CON / DEX / INT / WIZ (직업별 고정 스탯)
- 성장: 레벨업 → 스탯 기반 수치 자동 증가

### 전투 시스템
- 방식: 자동 전투 + 스킬 개입 혼합
- 몬스터: 비선공 (플레이어 선택 시 시작) / 선공 (자동 조우) 구분

### 스토리텔링
- 구조: 메인 퀘스트 / 지역 퀘스트 / 반복 퀘스트
- 원칙: 지역 진입 묘사, 몬스터 고유 설명, NPC 세계관 반영

## NPC 대사 관리 원칙

### 기본 규칙
- 모든 NPC 대사는 text_mmorpg_npc_dialogues 테이블에서 관리
- 코드에 대사 하드코딩 금지
- 대사 추가/수정 시 SQL 마이그레이션 파일로 관리

### dialogue_type 분류
- `default`: 첫 대화 (기본 스토리 대사)
- `quest_offer`: 퀘스트 제안 대사 (progress=0, 재대화)
- `in_progress_1~N`: 퀘스트 진행 중 재대화 (story_progress별)
- `completed`: 해당 막 완료 후 대화

### 대사 추가 방법
새 NPC 또는 대사 추가 시:
1. `migrations/NNN_npc_dialogue_추가내용.sql` 파일 생성
2. `INSERT INTO text_mmorpg_npc_dialogues` 실행
3. SQL Editor에서 실행 후 커밋

### 추후 에이전트 연동
- Max 플랜 전환 후 콘텐츠 라이터 에이전트가
  NPC 대사를 생성하여 DB에 직접 삽입
- 코드 수정 없이 콘텐츠 확장 가능
